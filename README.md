# circle-arc-to-polygon

Converting circle arc area given center point, inner radius, outer radius, start angle, stop angle, to a polygon, in geojson format, following the right hand rule

## Setup

```sh
git clone https://github.com/GPT-Engineer-App/circle-arc-to-polygon.git
cd circle-arc-to-polygon
npm i
```

```sh
npm run dev
```

This will run a dev server with auto reloading and an instant preview.
